<static> className(className) → {binder}

className binder

This function is a shortcut for using existence of element's className as boolean value when you bind it to a property.
Returns: binder
Arguments
className string
Since:
0.1
Source:
matreshka-binders.js, line 35
Examples

Usage

this.bindElement( 'myKey', '.my-element', MK.binders.className( 'blah' ) );
// same as
this.bindElement( 'myKey', '.my-element', { // no "getValue" and no "on" property
	setValue: function( v ) {
		$( this ).toggleClass( 'blah', v );
	}
});
this.myKey = true; // adds 'blah' class to '.my-element'
this.myKey = false; // removes 'blah' class from '.my-element'

Using "!" (not) statement

this.bindElement( 'shown', '.my-element', MK.binders.className( '!hide' ) );
// same as
this.bindElement( 'shown', '.my-element', { // no "getValue" and no "on" property
	setValue: function( v ) {
		$( this ).toggleClass( 'hide', !v );
	}
});
this.shown = true; // removes 'hide' class from '.my-element'
this.shown = false; // adds 'hide' class to '.my-element'

<static> innerHTML() → {binder}

innerHTML binder

By default if you pass html element to Matreshka#bindElement as second argument that doesn't match any defaultBinders item, the binding does nothing. But sometimes you want to change innerHTML without having possibility to retrieve value from the element. Matreshka.htmlp is created as simple binder function for this case and it reduces your code.
Returns: binder
Since:
0.1
Source:
matreshka-binders.js, line 14
Example

Usage

this.bindElement( 'myKey', '.my-element', MK.binders.innerHTML() );
// same as
this.bindElement( 'myKey', '.my-element', { // no "getValue" and no "on" property
	setValue: function( v ) {
		this.innerHTML = v;
	}
});
