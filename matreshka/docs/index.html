Matreshka documentation

Wellcome to Matreshka docs. You can find there explanations of what any method or property does. Some methods marked as deprecated. That means that these methods will be removed in next major version.

How to use Matreshka?

There are 3 cases how to attach Matreshka to your page.

IE9+ using jQuery

<script src="js/jquery.js"></script>
<script src="js/matreshka.min.js"></script>

IE9+ without jQuery

Matreshka doesn't require jQuery from version 0.1.0. If jQuery is not included, Matreshka will use own tiny DOM utilite called Balalaika.

<script src="js/matreshka.min.js"></script>

IE8

Pay attention that Balalaika works with modern browsers only. If you use Internet Explorer 8, you still need jQuery. Besides, Mateshka (Matreshka.Array) requires modern Array.prototype methods so you must use polyfill called es5-shim

<script src="js/es5-shim.min.js"></script>
<script src="js/jquery.js"></script>
<script src="js/matreshka.min.js"></script>

How to attach scripts as is?

If you're developer and if you'd like to run Matreska source files, keep order as described below.

<script src="node_modules/es5-shim/es5-shim.js"></script> <!-- Required for IE8 -->
<script src="node_modules/jquery/dist/jquery.js"></script> <!-- Required for IE8 -->
<script src="src/polyfills/classlist.js"></script> <!-- Required for IE8 and IE9 -->
<script src="src/xclass.js"></script>
<script src="src/balalaika.js"></script>
<script src="src/balalaika-plugins.js"></script>
<script src="src/matreshka-core.js"></script>
<script src="src/matreshka-binders.js"></script>
<script src="src/matreshka-object.js"></script>
<script src="src/matreshka-array.js"></script>
