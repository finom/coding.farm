Examples

Basic usage

new MK.Object;

Passing object

// creates Matreshka.Object instance with 2 enumerable keys (a, b)
new MK.Object({ a: 1, b: 2 });
// same as new MK.Object().jset({ a: 1, b: 2 });

Inheritance

var MyClass = Class({
	'extends': MK.Object,
	constructor: function() {
		// calls MK.Object constructor with same context and given arguments
		MyClass.parent.constructor( this, arguments );
	},
	method: function() {}
});

jset(key, value, evtOpts) → {mkObject}

Sets given property and adds key to enumerable list

This is the important method of Matreshka.Object insance that does two things:

1. Sets property.
2. Adds given key to enumerable list (that key enumerates via Matreshka.Object#each method and includes to object that returnes from Matreshka.Object#toObject and Matreshka.Object#toJSON) methods.

Returns: mkObject (self)
Fires: change change:KEY modify
Arguments
key string
value any
evtOpts <optional>
 		eventOptions
Source:
matreshka-object.js, line 242
Examples

Basic usage

this.jset( 'a', 1 ).jset( 'b', 2 );

Matreshka.Object#each

this.jset( 'a', 1 ).jset( 'b', 2 );
// set 'c' to 3 but do not add keys to enumerable list
this.set( 'c', 3 );
this.each( function( value, key ) {
	console.log( key, value ); 
});
// logs 'a' 1 and 'b' 2

Matreshka.Object#keys

this.jset( 'a', 1 ).jset( 'b', 2 );
// set 'c' to 3 but do not add keys to enumerable list
this.set( 'c', 3 );
console.log( this.keys() ); // logs [ 'a', 'b' ]

Matreshka.Object#toObject

this.jset( 'a', 1 ).jset( 'b', 2 );
// sets 'c' to 3 but not adds keys to enumerable list
this.set( 'c', 3 );
console.log( this.toObject() ); // logs { a: 1, b: 2 }

After using Matreshka.Object#jset(1) you can work with property as with regular property

this.jset( 'a', 1 ).jset( 'b', 2 ); // sets properties and adds to 'a' and 'b' enumerable list
this.set( 'a', 3 ); 
this.b = 4;
