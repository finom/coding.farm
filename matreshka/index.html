Why Matreshka?

Matreshka is the powerful framework that makes you forgetting about UI updates when you change application data. Matreshka gives you possibility to operate with a model without taking care about HTML.

Magic

Matreshka provides you magic which you only dreamed of: getters and setters, dependencies between one data from another data, data change events and more. It works even in Internet Explorer 8!

Clean HTML

There're many Javascript frameworks that create new language by extending HTML language. Developers that use these frameworks should change HTML code by adding new attributes and new syntax stuff. We love HTML and {{dont.like}} when application logic goes to HTML code. So one of the main advantages of Matreshka is keeping HTML to be just HTML.

Clear JavaScript

Matreshka is created by lazy developer for lazy developers who always forget to explain their code in comments, who likes to do less and get more, who doesn't like confusing abstractions, who likes to create clear and understandable code so anybody can get what is going on even if dosn't know Matreshka syntax.

Open-source and free

Matreshka is developing under MIT license. It means that Matreshka is open-source and completely free and you can do with it everything you want. The code is stored on Github as public repository so anybody can review it.
