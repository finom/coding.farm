Matreshka.js

The new way how you build your apps

No code in HTML with smart element bindings

You don't need to listen element event to change app property. You don't need to change element state manually when changing property.


this.bindElement( 'myKey', '.my-element' );

this.myKey = 'Be Awesome';

Awesome client side event emitter

You can even bind property change event. Isnt it awesome?

this.on( 'change:myKey', function( evt ) {
	alert( 'myKey value is ' + evt.value );
});

this.myKey = 'F... Yeah!'

Works even in IE8

Magic inside Matreshka gives possibility to use getters and setters.


this.define( 'myKey', {
	get: function() { ... },
	set: function( v ) { ... }
});

Build your powerful app using Matreshka

Matreshka uses classes for simple heritage.

var MyApp = Class({
	'extends': Matreshka,
	constructor: function() {
		this.initMK();
	}
});
