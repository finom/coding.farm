Why Matreshka?

Matreshka is the powerful framework that does you forgetting about element updating when you change your application data. Matreshka gives you possibility to operate with a model without taking care about UI.

Magic

Matreshka provides you magic which you only dreamed of: getters and setters, dependences of one data from another data, data change events and more. It works even in IE8!

No dirty HTML

There're many Javascript frameworks that create new language by extending HTML. Developers that use these frameworks should change HTML code by adding new attributes and new syntax stuff. We love HTML and {% raw %}{{dont.like}}{% endraw %} when application logic goes to HTML code. So one of the main advantages of Matreshka is keeping HTML to be just HTML.

Get semantic code

Methods and properties of Matreshka are named so that anybody can understand what's going on and what the method does.

Open-source and free

Matreshka is developing under MIT license, it means that Matreshka is open-source and completely free and you can do with it averything you want.
