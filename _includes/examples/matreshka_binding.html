Element binding

The next 3 examples show you how to bind data to UI. More info about bindElement method you can find in on documentation page.

{% highlight javascript %}var mk = new MK(); // same as new Matreshka; mk.bindElement( 'x', '.my-select', { on: 'change', getValue: function() { return this.value; }, setValue: function( v ) { this.value = v; } }); mk.x = 3; {% endhighlight %}
