.addDependency method

.addDependency method allows property to be dependent from other properties.

Get more on documentation page

{% highlight javascript %}var mk = new MK(); mk.on( 'change:a', function() { $( '.output' ).html( 'a = ' + this.a ); }); // a depends from b and c as their sum mk.addDependency( 'a', 'b c', function() { return this.b + this.c; }); mk.b = 11; mk.c = 22; {% endhighlight %}
