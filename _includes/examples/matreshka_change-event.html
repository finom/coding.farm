change: event

Matreshka allows you to listen property change event. When you're adding listener for change:%PROPERTY_NAME% event, Matreshka creates setter for this property. It means that you can change the property directly, without calling methods such as .set.

{% highlight javascript %}var mk = new MK(); mk.on( 'change:x', function() { var text = 'x changed to ' + this.x; $( '.output' ).html( text ); }); mk.x = 5; {% endhighlight %}
